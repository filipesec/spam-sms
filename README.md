# SMS Spam
Trabalho de Reconhecimento de Padrões para classificação de mensagens SMS em spam e não spam utilizando Logistic Regression

## Estrutura do Projeto
spam-sms/
├── src/
│ ├── eda.py # Análise exploratória
│ ├── processing.py # Pré-processamento e TF‑IDF
│ ├── train_model.py # Treinamento do modelo
│ └── evaluation.py # Avaliação e métricas
├── data/
│ └── spam.csv # Dataset original
├── outputs/ # Gráficos gerados
├── main.py # Pipeline principal
├── requirements.txt # Dependências do projeto
└── README.md

## Pré‑requisitos

- Ambiente virtual Python (`venv`)
- pandas
- matplotlib

## Para Instalar Dependências:

- pip install -r requirements.txt
